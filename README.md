🎨 ArtCraft
Transforme o navegador em um verdadeiro estúdio de arte digital.

🚀 Sobre o Projeto
O ArtCraft é um editor de pintura digital completo que roda diretamente no navegador, inspirado em softwares profissionais como Ibis Paint.

Ele foi desenvolvido com foco em oferecer uma experiência real de criação artística, combinando performance, interface moderna e ferramentas avançadas — tudo sem necessidade de instalação.

✨ Funcionalidades
🎨 Canvas
Criação de telas personalizadas
Presets (Full HD, A4, 4K, etc.)
Zoom e navegação fluida
Renderização em tempo real
🖌️ Ferramentas
Pincel
Borracha
Borrar (Smudge)
Desfoque (Blur)
Preenchimento (Bucket)
Conta-gotas
Texto
Seleção (Laço e Varinha Mágica)
Transformação
🧱 Camadas
Criar, duplicar e excluir camadas
Controle de opacidade
Modos de mesclagem
Merge de camadas
Organização visual
🎨 Cores
Roda de cores (Color Wheel)
Ajustes com sliders
Input HEX
Paleta de cores e histórico
🧰 Pincéis
Biblioteca de pincéis
Importação de brushes
Preview em tempo real
Configurações avançadas
💾 Arquivos
Abrir imagens
Exportar arte
Sistema de salvamento
🖥️ Interface
O ArtCraft possui uma interface inspirada em softwares profissionais:

Topbar com ações principais (undo, redo, salvar)
Sidebar esquerda com ferramentas
Painel direito com camadas e pincéis
Painel inferior com cores e configurações
Tela inicial para criação de projetos

🎯 Objetivo
O ArtCraft busca transformar o navegador em um ambiente completo de criação artística, permitindo que qualquer pessoa desenhe, edite e experimente sem precisar de softwares pesados.
